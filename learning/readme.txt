learning submodule of optimalBC

The sequence of experiments
1. baseline - the basic learning with knn krr gpr with DMP
2. passiveAL - the passive sampling based active learning pipeline
3. activeAL - the active optimization based active learning pipeline
4. neural_networks - the very simple NN approach
5. baseline_spl - the b-spline weight predicting
6. baseline_spl_via - via point prediction with spline interpolating
7. activeAL_spl_via - via point active learning
8. acceleration_learning - learning the controller dynamics
